# 抖店-多功能脚本

 请使用谷歌浏览器安装油猴拓展后访问抖店。

## 反馈建议
#### 加微信:YANG706597125
![screenshot](https://s2.loli.net/2022/01/20/keFMgb3iKC5lPr1.jpg)


## 功能：

- 一键下载订单信息，导出csv。
- 批量显示隐藏的发货信息。
- 一键复制订单信息，发货信息是隐藏状态，复制的是隐藏的，先显示隐藏信息。
